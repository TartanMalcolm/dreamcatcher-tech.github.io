Contributing

Ambient Attribution based ways to contribute

Write Code

Making code contributions is open to anybody. Please contact us if you are having trouble getting your environment set up, or have any questions.

Get Started

Raise Issues

Tell us what is wrong with the apps, or with the codebase. This is valuable to us. Even helping clarify issues that others have written is valuable.

Get Started

Fund Issues

You can put up money directly thru gitcoin or any other system to pay money for people to close issues you choose. We will attribute you the value of the issue you closed between you and the people who did the work.

Get Started

Fund the organization

Funding is managed through an Aragon DAO and is tracked with Inversion Tokens, with symbol NVN This is a bonded curve token and is open to accredited investors only.

Get Started

Write Documenation

Good clear documentation is essential for onboarding. Please help us improve what we have, and suggest anything that might need changing.

Use the software

If you find something useful to use the software for, please let us know. We would love to know what problems you had getting set up, and what
